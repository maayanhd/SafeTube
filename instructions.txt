הסבר כללי:
 
הפרוייקט מורכב מ-3 שרתים - שרת backend ב-Node.js, שרת frontend בReactJS ושרת http בפייתון לצורך חישובים ומתן ציון על סרטון/פלייליסט.

הוראות הפעלה: 

‎‏‎‏1. התקן spark וhadoop (הגרסא העדכנית ביותר) על המחשב (דורש הגדרת environment variables (מצורפות תמונות בתיקייה Environment Variable Images הנמצאת בתיקייה הראשית - SafeTube).
‎‏2. נווט ל- SafeTube\client בterminal
  2.1 הרץ npm install 
  2.2 הרם את שרת הfront עם npm start
3. נווט ל- SafeTube\Server  בterminal
 3.1 הרץ npm install 
 3.2 הרם את שרת ה back עם npm start (עדיפות לשימוש ב-terminal של visual studio code - למקרה שport 3000  תפוס)
4. נווט ל- SafeTube\scoring_videos והרץ את Video_Scoring_System_metadata.py ב-IDE לבחירתך וחכה לחיווי כי השרת רץ (יופיע בקנוסול - server is live).  
5. שים סרטון/פלייליסט באתר שנפתח וראה את התוצאות.
